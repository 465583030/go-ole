go-ole
======

WHATS:
------

  Go bindings for Windows COM

INSTALL:
--------

  To experiment with go-ole, you can just compile and run the example
  program:

    go get github.com/mattn/go-ole
    go get github.com/mattn/go-ole/oleutil
    cd /path/to/go-ole/example/excel
    go run excel.go
